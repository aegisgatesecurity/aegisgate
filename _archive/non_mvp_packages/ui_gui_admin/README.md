# AegisGate GUI Administrator Interface

This is the Electron-based GUI for AegisGate Chatbot Security Gateway.

## Features

- Dashboard with real-time monitoring
- Traffic inspection logs
- Compliance framework management
- Certificate management
- Settings configuration

## Getting Started

1. Navigate to ui/gui-admin
2. Run: npm install
3. Run: npm start

## Compatibility

- Windows 10+ (64-bit)
- macOS 10.14+ (Intel & Apple Silicon)
- Linux (Debian/Ubuntu 20.04+)
