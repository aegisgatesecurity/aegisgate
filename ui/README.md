# AegisGate Dashboard

A modern, responsive web interface for the AegisGate Security Gateway.

## Features

- Real-time metrics display
- Audit trail viewing
- Compliance status monitoring
- Policy management ( forthcoming )
- Certificate management ( forthcoming )

## Directory Structure

### ui/
- frontend/
  - index.html
  - css/style.css
  - js/dashboard.js

## Usage

1. Start the AegisGate Security Gateway
2. Access the dashboard at: http://localhost:8443/dashboard
3. View real-time metrics and audit logs
4. Monitor system status
