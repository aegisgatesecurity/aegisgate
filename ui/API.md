# AegisGate API Documentation

## Overview

The AegisGate API provides endpoints for dashboard integration, monitoring, and management.

## Endpoints

### Metrics
GET /api/v1/metrics
- Returns current system metrics

### Audit Log
GET /api/v1/audit/log
- Returns recent audit log entries

### Status
GET /api/v1/status
- Returns system status information
